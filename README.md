Build
-----

To build Smith, you'll need [.NET Core SDK][dotnet] 5.0 or later.

```
$ dotnet build --configuration Release
```

(replace `Release` with `Debug` for debug build)

Project History
---------------

Smith is forked from [egram.tel][], a Telegram client.
